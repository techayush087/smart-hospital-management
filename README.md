# 🏥 Smart Hospital Appointment & Patient Management System

A modern and responsive **Hospital Appointment & Patient Management System** built using **Angular**, **TypeScript**, and **NgRx**. The application streamlines hospital operations by enabling secure authentication, appointment scheduling, patient record management, and administrative functionalities through an intuitive user interface.

## 🚀 Live Demo

🔗 https://appointment-system-advanced.vercel.app

---

## ✨ Features

- 🔐 Role-Based Authentication & Authorization
- 👨‍⚕️ Doctor Search and Filtering
- 📅 Appointment Booking, Rescheduling & Cancellation
- 👤 Patient Dashboard & Medical Records
- 📋 Prescription & Consultation Management
- 📊 Admin Dashboard
- 🔔 Notification System
- ⚡ State Management using NgRx
- 📱 Responsive User Interface

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/techayush087/smart-hospital-management.git
```

### Navigate to the project

```bash
cd smart-hospital-management
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
ng serve
```

Open your browser and visit:

```
http://localhost:4200
```
